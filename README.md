# Unity Manager: Gönüllü Yönetim Paneli

Unity Manager, gönüllü organizasyonların üye yönetimini, görev takibini ve toplantı planlamasını dijitalleştiren, terminal tabanlı bir yönetim yazılımıdır.

## Öne Çıkan Özellikler

- Üye Yönetimi: Yeni gönüllüleri sisteme kaydetme.
- Görev Takibi: Görev atama, durum güncelleme (Devam Ediyor / Tamamlandı).
- Gelişmiş Filtreleme: Sorumlu kişiye veya görev durumuna göre dinamik arama.
- Toplantı Planlayıcı: Tarih ve saat bazlı toplantı organizasyonu.
- İstatistik Raporlama: Başarı oranı ve aktif üye sayılarını anlık analiz etme.
- Veri Kalıcılığı (Persistence): Tüm veriler JSON formatında güvenli bir şekilde saklanır.

## Kullanılan Teknolojiler

* Dil: Python 3.14.3
* Veri Yapısı: JSON (Dosya tabanlı veritabanı mantığı)
* Modüller: `json`, `datetime`
